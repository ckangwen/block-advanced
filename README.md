## 介绍

本项目基于vue和element-ui，总结一些常用的应用场景，旨在提升代码质量，规范开发流程。



## 快速开始

src目录下是具体的项目文件，各自之间相互独立，没有关联。

如果要启动任意一个项目，只需执行package.json中相应的的scripts命令。

```bash
# 安装依赖
yarn install

# 启动指定的服务
yarn run login
```

启动完成后，浏览器访问 [http://localhost:8080](http://localhost:8080) 即可预览。
